# Automated DevSecOps CI/CD, QA & Project Management Pipeline

Built an end-to-end DevSecOps pipeline integrating GitLab and Jenkins to automate developer-to-QA workflows with continuous integration and deployment. Implemented structured project tracking and issue management using Redmine to simulate a complete software development lifecycle (SDLC), including task assignment, ticket tracking, and workflow visibility.

Integrated multi-layer security and quality validation using SonarQube, Snyk, Gitleaks, and OWASP-based scanning to detect vulnerabilities early in the development lifecycle.

Added automated testing pipelines using Cypress for end-to-end testing, Newman for API validation, and JMeter for performance and load testing. Designed a containerized multi-server infrastructure using Docker, isolating CI/CD, project management, security scanning, and application services for scalability, modularity, and maintainability.
