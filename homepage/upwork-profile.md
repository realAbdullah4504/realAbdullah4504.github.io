## Profile Overview

I build backend systems and cloud infrastructure that scale from day one.

Over the past 5 years, I've delivered production-grade platforms — distributed processing systems handling 50+ data sources, event-driven AWS architectures, and end-to-end CI/CD pipelines for SaaS platforms. My systems are built for real traffic, real failures, and real operations.

**What I Build**
• Distributed backend services & APIs (Node.js, Python, REST, GraphQL)
• Event-driven processing systems with queues, workers, and background jobs
• AWS infrastructure — ECS Fargate, Lambda, SNS/SQS, VPC, IAM, CloudFormation
• CI/CD pipelines with security scanning, automated testing, and zero-downtime deployment
• Observability stacks — distributed tracing, Prometheus, Loki, structured logging
• Multi-tenant SaaS platforms with authentication, payments, and real-time features

**Engineering Approach**
I design from the load in, not the tools out. Every system includes explicit failure handling (idempotency, dead-letter queues, circuit breakers), observability at every boundary, and infrastructure as code. Production readiness is non-negotiable from the first commit.

**Proof**
Distributed scraping platform: 20 parallel workers, ~5-minute batch cycles across 50+ government portals.
Cloud infrastructure: Single-command environment provisioning via CloudFormation/Terraform.
Observability: Unified tracing + metrics stack cutting mean time to detection on production incidents.
