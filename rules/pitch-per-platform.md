# Pitch Per Platform

## Source of Truth

Projects are the source of truth, but they do **not** directly generate everything linearly. Instead, they feed multiple parallel layers — capabilities (identity), services (offers), and case studies (proof) — which then collectively form pricing, portfolio, and all public-facing profiles like Upwork, Fiverr, CV, and README.

## Platform Transformations

Transform the same underlying content differently for each platform:

- **CV:** 2–3 bullets, impact-focused
- **Upwork:** Problem → Solution → Outcome
- **Portfolio:** Full story + architecture
- **LinkedIn:** Narrative + positioning

## Project Value Tiers

System-heavy backend projects — **High Value**  
AI + automation projects — **Very High Value** (if framed right)  
React UI / Figma conversions — **Supporting Value**  
AI-generated UI + Supabase + custom logic apps — **Hidden Gold**  
RAG systems + deployments — **Senior Signal** (if positioned correctly)

## AWS Services

A broad range of AWS services used across infrastructure and deployment work:

- **Compute & Containers:** EC2, Lambda, ECS, EKS, Elastic Beanstalk
- **Storage & Databases:** S3, RDS, DynamoDB, DocumentDB
- **Networking:** VPC, Elastic Load Balancing, Route 53, PrivateLink
- **Monitoring & Logging:** CloudWatch, X-Ray, CloudTrail, Systems Manager
- **Messaging & Events:** SQS, SNS, EventBridge
- **CI/CD:** CodeBuild, CodeDeploy, CodePipeline
